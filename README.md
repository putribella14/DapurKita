# Dapur Kita
Selamat datang di Dapur Kita, sebuah website resep makanan yang dirancang dengan tampilan modern, responsif, dan mudah digunakan. Website ini dibuat sebagai Proyek Tambahan Kelas Industri untuk memenuhi tugas pengembangan website.

# Deskripsi
Dapur Kita merupakan website yang menyediakan berbagai resep makanan yang dapat membantu pengguna menemukan inspirasi masakan dari berbagai negara dengan mudah. Website ini memiliki tampilan yang sederhana namun menarik sehingga nyaman digunakan pada komputer maupun perangkat seluler.

# Tujuan Website
* Memudahkan pengguna mencari resep makanan.
* Menyediakan informasi resep yang lengkap dan mudah dipahami.
* Menerapkan kemampuan HTML, CSS, dan JavaScript dalam pembuatan website.
* Memenuhi tugas Proyek Kelas Industri.

# Cara Menjalankan Website
1. Download atau clone repository proyek.
2. Ekstrak file apabila masih dalam format ZIP.
3. Buka folder **Dapur Kita**.
4. Jalankan file **index.html** menggunakan browser seperti:

   * Google Chrome
   * Microsoft Edge
   * Mozilla Firefox
5. Website siap digunakan.

# Cara Penggunaan
1. Buka halaman utama website.
2. Pilih menu navigasi yang tersedia.
3. Gunakan fitur pencarian untuk menemukan resep.
4. Pilih kategori makanan sesuai keinginan.
5. Klik salah satu resep untuk melihat informasi lengkap.
6. Aktifkan Dark Mode apabila tersedia.
